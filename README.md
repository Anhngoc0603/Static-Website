# Static-Website
code giống môn WEBDEV trước á mn copy, chuyển đổi khi cho vào angular nha
Mn có thể clone về máy để xem web như nào nha vẫn chưa hoàn chỉnh á nên sẽ có thay đổi, mn để ý nhe  nào hoàn thiện N báo/ mn thấy thiếu hoặc sửa được thì sửa cx đc nha 
Repo chính: Anhngoc0603/SNEAKER_FINALWEBDEV

+ Lần đầu clone về máy. Chỉ cần làm 1 lần duy nhất:
git clone https://github.com/Anhngoc0603/SNEAKER_FINALWEBDEV.git
cd SNEAKER_FINALWEBDEV

+ Trước khi sửa bất kỳ file nào, luôn kéo bản mới nhất về:
git pull origin main

+ Chạy 3 lệnh vàng này để đẩy code lên GitHub:
git add .
git commit -m "mô tả ngắn gọn phần bạn sửa"
git push origin main

+ Kiểm tra nhánh đang đứng
git branch
